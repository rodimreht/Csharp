본 소스는 KISA에서 제공하는 C 버전의 SEED 암호화 코드를
C# 버전으로 마이그레이션 한 것입니다.

SEED 암호화 알고리즘을 이해하고 만든 것이 아니라,
C 소스를 그대로 C#에서 컴파일되게끔만 수정한 것이라서
C#적인 측면에서는 비효율적인 코드가 들어가 있을 수도 있습니다:D

SEED 클래스의 Encrypt/Decrypt 메소드는 특별한 입력체크를 하게 되어 있지는 않습니다...
잘못된 암호문을 입력할 경우, 런타임 오류가 발생합니다:D

SEED 암호화는 16바이트 대칭키 암호화로, 16바이트 단위로 데이터를 암호화합니다.
16바이트보다 적은 바이트의 원문은 암호화된 후에는 16바이트로 출력이 됩니다.

적용한 프로젝트에서 상대측 자바 SEED 코드에서 CBC-PAD 방식으로 암호화를 수행하고 있었기 때문에,
SEED 클래스에도 원본 C 소스 코드에는 없는 CBC-PAD 방식이 추가되어 있습니다.
일단 그 부분을 삭제하지는 않고 cbcPad 인수를 받아 분기할 수 있도록 수정했습니다.

CBC 방식의 초기값 IV는 null을 사용하고 있습니다. 다른 값을 사용할 경우에는 수정하여야 합니다.

원본 C 소스에서 사용하는 방식이 No Padding의 ECB 방식입니다.
이 방식과 CBC-PAD 방식을 제외한 다른 방식은 본 C# 소스에는 적용되어 있지 않습니다.


KISA에서 C 소스 자체를 공개하고 있으므로 큰 문제가 없을 거라 판단,
본 소스는 데브피아(www.devpia.com) C# 자료실에만 업로드합니다.
문제가 될 경우, 데브피아의 자료는 자체 삭제하겠습니다.

소스에 주석은...당연히 없습니다! C 소스 그대로 복사한 거니까요! :D
소스에 대한 문의는...받지 않습니다:D 저도 암호화 알고리즘은 몰라요:D
CBC-PAD 관련해서 추가한 내용은 Encrypt/Decrypt 메소드의 cbcPad 인수를 참조하시면 됩니다.
