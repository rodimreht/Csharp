(주)넷츠 사내 메신저 프로그램

- (주)넷츠 정규직 및 임시/계약직이면 별도의 등록 절차 없이 누구나 이 프로그램을
  바로 사용하실 수 있습니다.

- (주)넷츠 사내 의사소통 이외의 목적으로는 사용하실 수 없으며 외부로 소스코드 및
  바이너리를 무단 유출하는 경우에는 법적 처벌을 받을 수도 있습니다.

- Copyright(C) 2009, 넷츠에 모든 저작권이 있습니다.



[실행 환경]

- Microsoft Windows XP SP2 이상(Vista와 Windows Server 2003 까지는 테스트되었습니다만...)

- Microsoft .NET Framework 2.0 이상이 설치되어 있어야 합니다. (SP1 권장)



[주요 기능]

- 조직도 표시
- 1:1 대화
- 회의실 기능(다중 채팅)
- 이미지/파일 첨부 기능(1MB 이하)
- 대화 채널(TCP 80 포트(HTTP) 사용)
- 고속 파일 전송 기능(1MB 이상; TCP 6421 포트 사용)
- 서버 경유 파일 전송 기능(방화벽/사설망의 경우; TCP 443 포트(SSL) 사용)
